# HocusFocus

특정 애플리케이션에서 활동한 누적 시간을 기록하는 Windows 데스크톱 앱입니다.

## 주요 기능

- **앱별 시간 추적**: 사용자가 지정한 애플리케이션의 사용 시간만 기록
- **유휴 감지**: 지정한 시간(기본 5분) 이상 입력이 없으면 자동으로 시간 기록 중지
- **포커스 감지**: 추적 대상 앱이 아닌 다른 창을 보고 있으면 시간 증가 안 함
- **일별 통계**: 매일 집중한 시간이 자동으로 집계됨
- **주간 차트**: 최근 7일간의 집중 시간을 차트로 시각화
- **시스템 트레이**: 백그라운드에서 실행되며 트레이 아이콘으로 빠른 제어 가능

## 스크린샷

앱을 실행하면 다음과 같은 대시보드가 표시됩니다:
- 오늘의 총 집중 시간
- 주간 집중 시간 차트
- 앱별 사용 시간 상세
- 추적 앱 관리
- 실행 중인 앱 목록

## 시스템 요구사항

- Windows 10/11
- .NET 8.0 Runtime

## 빌드 방법

```bash
dotnet build
```

## 실행 방법

```bash
dotnet run
```

또는 빌드 후 `bin\Debug\net8.0-windows\HocusFocus.exe` 실행

## 사용 방법

1. **앱 실행**: 프로그램을 실행하면 대시보드 창이 나타납니다.

2. **추적할 앱 추가**: 
   - 오른쪽 "실행 중인 앱" 목록에서 추적하고 싶은 앱을 찾습니다.
   - "+ 추가" 버튼을 클릭하여 추적 목록에 추가합니다.

3. **시간 기록**:
   - 추적 대상 앱을 사용하면 자동으로 시간이 기록됩니다.
   - 유휴 상태거나 다른 앱을 사용하면 시간이 증가하지 않습니다.

4. **유휴 시간 설정**:
   - 설정에서 유휴 감지 시간을 1~30분 사이로 조절할 수 있습니다.

5. **백그라운드 실행**:
   - 창을 최소화하거나 닫으면 시스템 트레이로 이동합니다.
   - 트레이 아이콘을 더블클릭하면 대시보드가 다시 나타납니다.
   - 트레이 아이콘 우클릭으로 일시정지/재개, 종료 가능

## 데이터 저장

- 모든 데이터는 `%AppData%\HocusFocus\data.json`에 저장됩니다.
- 30초마다 자동 저장되며, 앱 종료 시에도 저장됩니다.
